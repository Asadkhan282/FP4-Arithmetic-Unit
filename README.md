# FP4-Arithmetic-Unit

High-performance 4-bit floating-point (FP4) arithmetic unit optimized for Xilinx Artix-7 (xc7a35t). Achieved 200 MHz timing closure with 1.460 ns WNS for AI hardware acceleration.

---

### 📊 Performance & Timing Verification
The design was verified with a full timing analysis to ensure high-speed reliability.
* **Worst Negative Slack (WNS):** 1.898 ns (Met)
* **Total Failing Endpoints:** 0
